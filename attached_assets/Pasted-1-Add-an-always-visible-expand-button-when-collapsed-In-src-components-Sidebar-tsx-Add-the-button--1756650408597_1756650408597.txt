1) Add an always-visible expand button when collapsed
In src/components/Sidebar.tsx

Add the button near the end of the <aside> (just before </aside>), outside the flyout block:

{collapsed && (
  <button
    className="railExpand"
    onClick={() => setCollapsed(false)}
    title="Expand"
    aria-label="Expand navigation"
  >
    <Lucide.ChevronRight size={18} />
  </button>
)}


That gives you a clear control to open the rail, even if hover-flyout is off.
