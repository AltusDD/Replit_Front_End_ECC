Copy/paste the two tiny patches below. They force the sidebar to behave exactly as we want when collapsed (no titles/labels), and they lock in the icon colors (parents gold, children alt-blue) so your theme can’t override them. It also makes the expand button look right.

1) Add this inline style block at the very end of <aside> in src/components/Sidebar.tsx

(Place it just before </aside> so it wins over global CSS.)

{/* ——— Local, sidebar-scoped overrides to beat global/visited rules ——— */}
<style
  // loads after global CSS; scoped to .sidebar; safe & surgical
  dangerouslySetInnerHTML={{
    __html: `
    /* kill purple visited links & any theme link colors inside sidebar */
    .sidebar a,
    .sidebar a:link,
    .sidebar a:visited,
    .sidebar a:active,
    .sidebar .nav-row,
    .sidebar .nav-row:visited {
      color: var(--ecc-text) !important;
      text-decoration: none !important;
    }

    /* collapsed = icon-only */
    .sidebar.collapsed .section-title,
    .sidebar.collapsed .label,
    .sidebar.collapsed .expand {
      display: none !important;
    }

    /* highest specificity so icons keep our colors */
    .sidebar .nav-row .icon.parent { color: var(--ecc-gold) !important; }
    .sidebar .nav-row .icon.child  { color: var(--ecc-dim)  !important; }
    .sidebar .flyout .icon.parent  { color: var(--ecc-gold) !important; }
    .sidebar .flyout .icon.child   { color: var(--ecc-dim)  !important; }

    /* fly-out always bounded */
    .sidebar .flyout { max-width: var(--ecc-flyout-w, 280px) !important; }
  `,
  }}
/>

2) Append these rules to the end of src/styles/_ecc-override.css

(They reinforce the collapsed state and fix the rail expand button visual.)

/* Make sure anchors never go purple/blue in the sidebar */
.sidebar a,
.sidebar a:link,
.sidebar a:visited,
.sidebar a:active {
  color: var(--ecc-text) !important;
  text-decoration: none !important;
}

/* Collapsed should be icon-only — titles/labels must hide */
.sidebar.collapsed .section-title,
.sidebar.collapsed .label,
.sidebar.collapsed .expand {
  display: none !important;
}

/* Icon color discipline  */
.sidebar .nav-row .icon.parent { color: var(--ecc-gold) !important; }
.sidebar .nav-row .icon.child  { color: var(--ecc-dim)  !important; }

/* Rail expand button look/placement (only visible when collapsed) */
.sidebar .railExpand{
  position: absolute;
  bottom: 16px; left: 50%;
  transform: translateX(-50%);
  width: 38px; height: 38px;
  display: inline-flex; align-items: center; justify-content: center;
  border: 1px solid var(--ecc-seam);
  background: var(--ecc-hover);
  color: var(--ecc-text);
  border-radius: 999px;
  box-shadow: 0 6px 16px rgba(0,0,0,.25);
  cursor: pointer;
  transition: transform .15s ease, background .15s ease, box-shadow .15s ease;
  z-index: 70;
}
.sidebar .railExpand:hover{ transform: translateX(-50%) scale(1.05); }
.sidebar:not(.collapsed) .railExpand{ display:none; }  /* hide when expanded */
